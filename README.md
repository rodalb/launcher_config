# Nordemon Launcher - Configuração

Este repositório contém apenas os arquivos de configuração para o launcher Nordemon.

## Arquivos

- `client_version.txt` - Versão atual do cliente
- `client_download_url.txt` - URL de download do cliente completo

## Download do Launcher

Baixe o launcher oficial em: [Releases](https://github.com/rodalb/launcher_config/releases)

## Para Atualizar o Cliente

1. Edite `client_version.txt` com a nova versão
2. Faça upload do novo cliente ZIP na release correspondente
3. Atualize `client_download_url.txt` com a nova URL
4. Commit e push das alterações

O launcher irá detectar automaticamente a nova versão e baixar o update.
